## Deployment

To deploy this project run

```bash
  cd build/
  cmake --build . or make
  ./ownscript
```

if cannot run please re-create the build folder

```bash
  rm -rf build
  mkdir build
  cd build/
  cmake ..
  cmake --build . or make
  ./ownscript
```
